How to play bee+

In each of seven stages, make the longest word possible using all of the available letters.

The game starts with four letters. In each stage, one letter is added, until you have seven letters. After that, one letter is removed in each stage, until you're back to four letters.

You don't need to complete the stages in order. If you get stuck, you can go forward or back. Once you've finished all the stages, click the check mark to complete the puzzle.
